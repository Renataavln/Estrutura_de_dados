# Ordenação por Comparação 2 - Merge Sort e Quick Sort

## Como compilar

Para compilar é necessário rodar o comando ```make``` no terminal.

## 1. Entrada manual (via terminal)

Após compilar, execute o programa e insira o tamanho do vetor e os elementos manualmente:

```./pratica2```

### Exemplo de interação:

```
Digite o tamanho do vetor: 4
Digite os 4 elementos:
10 5 8 2
```

## 2. Entrada via Arquivo .txt

Você pode criar um arquivo de texto onde a primeira linha é o tamanho do vetor e as seguintes são os elementos. Para rodar o teste sem precisar digitar, use o redirecionamento:

```./pratica2 < entrada.txt```

Para testar todos os casos testes, rode:

```
./pratica2 < testes/teste1.txt
./pratica2 < testes/teste2.txt
./pratica2 < testes/teste3.txt
./pratica2 < testes/teste4.txt
```
